# 🌍 Voyato - Traveling App

Voyato is a modern travel planning and destination discovery platform designed to simplify trip management and enhance travel experiences. The application enables users to explore destinations, organize travel plans, manage itineraries, and access personalized travel recommendations through a user-friendly interface.

## ✨ Features

✈️ Trip Planning & Management
🗺️ Destination Discovery
📅 Travel Itinerary Organization
📍 Location-Based Recommendations
❤️ Favorite Destinations
🔐 User Authentication
☁️ Firebase Integration
📱 Responsive Design
